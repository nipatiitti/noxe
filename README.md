# NoXe
 
* [Website](https://www.noxe.tk/)
Used to make notes. No Extra stuff only plain simple notes

## Getting Started

1. Clone git repo
2. Run:

```
npm install
```


### Prerequisites

Node
Npm/yarn

## Deployment

```
npm run build

or

yarn build
```

## Built With

* [React](https://reactjs.org/) - View library
